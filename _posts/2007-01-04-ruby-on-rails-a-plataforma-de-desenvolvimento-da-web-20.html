Eis a web 2.0.

Para acompanhar essa tendência, Ruby on Rails.

Ruby on Rails é o framework para desenvolvimento de aplicações web para a linguagem Ruby. Traz consigo, diversas classes de apoio (Helpers), para trabalhar com ajax, javascript, widgets (controles), efeitos visuais etc.

O slogan mas utilizando para Ruby on Rails é: don't tell, show (Não fale, mostre).

Se você quiser saber mais sobre essa excelente plataforma de desenvolvimento para web, visite esses sites:

http://www.rubyonbr.org (Comunidade brasileira de Ruby on Rails)
http://www.rubyonrails.org (Site oficial da plataforma)
http://api.rubyonrails.org (Documentação oficial da api do Ruby on Rails)
