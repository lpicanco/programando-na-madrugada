Script SQL simples para remover os registros duplicados de uma tabela:


DELETE FROM Links
WHERE ID NOT IN
(
	SELECT MAX(ID)
	FROM Links
	GROUP BY url
)

Removendo os registros com urls duplicadas da tabela Links.
